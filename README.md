# Flutter Student CRUD App

A Flutter CRUD app for managing student records using SQLite for local storage and Firebase for favorite records.

## Features

- Local SQL database using `sqflite`
- Create, Read, Update, Delete (CRUD) operations
- Search by name, address, or exact ID
- Edit records using a dialog box
- Form validation for name, age, and address
- Mark favorite records and store them in Firebase Firestore
- Firebase Anonymous Authentication
- Works for local CRUD even if Firebase is not configured

## Tech Stack

- Flutter
- Dart
- SQLite (`sqflite`)
- Firebase Auth
- Cloud Firestore
